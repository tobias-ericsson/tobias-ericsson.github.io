PM2 provides an easy way to manage and daemonize applications (run them as a service).

sudo npm install pm2 -g

pm2 start node-server.js

pm2 startup ubuntu

sudo env PATH=$PATH:/usr/bin pm2 startup ubuntu -u tibbe
